# 通信特点

串行同步半双工

1.同步(有时钟线) + 串行

2.半双工(只有一条数据线进行收发)

3.有应答位(应答机制)

4.可一主多从,也可多主多从

5.高位先行

6.两条线就能实现:SCL,SDA

7.当前地址指针在每一次读或写后都会加1

8.I2C协议的通信端口都要配置为开漏模式,且外接一个上拉电阻(上拉电阻一般为)

# 数据帧

起始位 + 数据帧 + 应答位 + 停止位

时序表现:

起始位: SCL高电平期间,SDA出现下降沿

停止位: SCL高电平期间,SDA出现上升沿

在该两位以外

在SCL高电平期间, 稳定的SDA将会被认为是有效信号
在SCL低电平期间,SDA能进行高低电平切换

应答位:接收方在数据帧发送完后发出,以表示是否收到/可继续

# 时序

在与模块进行通信时,需要知道模块的器件地址(7位或10位,7位居多)和寄存器地址

一般写一字节数据的时序

起始位 + 7位 + 1位读写位(写) + 接收方应答位+寄存器地址位(8位)+接收方应答位+数据位(8位)+停止位

一般读一字节数据的时序

起始位 + 7位器件地址位 + 1位读写位(写) + 接收方应答位 + 寄存器地址位(8位)+接收方应答位 + 起始位 + 7位器件地址位+1位读写位(读)+接收方应答位+等待接收方发送数据 + 不应答 + 停止位
