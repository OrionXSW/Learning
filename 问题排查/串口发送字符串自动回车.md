/***********************************************
*函数名    :	Serial1_Receive_String
*函数功能  :	串口发送字符串
*函数参数  :	uint8_t *str
*函数返回值:	void
*函数描述  :
*引脚情况  :	PA9 PA10    TX/RX
***********************************************/
void Serial1_Receive_String(uint8_t str[])
{
    uint8_t i = 0;
    do{
        str[i] = Serial1_Receive_Byte();
        i++;
    }while(str[i-1] != '\r');
    str[i-1] = '\0';

}

这玩意,注:当你在串口软件上输入空格时,他的符号为 \r\n

所以当你因为检测到回车符时暂停输入,再次发送一些字符串时他会自动回车,就是因为\n没被消化掉
