# BKP

备份寄存器,存储的,由VBAT单独供电,只要有这个供电数据就不会丢失,VBAT通常接上纽扣电池
来存用户应用程序数据的
规格:对于中容量和小容量的 BKP为20字节
    对于大容量和互联型的 一般为84字节

BKP还有RTC部分

# 特性

当VDD主电源掉电时,后备区域仍然可以由VBAT的备用电池供电
而VDD有电时,后备区域使用VDD供电,以此来节省电池电量

# RTC

Real Time Clock 实时时钟 一个独立的定时器,为系统提供时钟和日历功能
RTC及时钟配置系统处于后备区域,系统复位时数据不清零
注:**要保证给到RTC计数器的时钟频率为1Hz**
